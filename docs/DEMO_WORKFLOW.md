# OncoEvidence Auditor Demo Workflow

## Verify the project

Run smoke tests, regenerate example reports, and run the readiness check before demoing.

## Demo cases

1. OIP5 in GBM: strong dependency but broad-essentiality and weak patient support.
2. ERBB2 in gastric cancer: therapeutic biomarker relevance without strong DepMap dependency.

## Main demo point

OncoEvidence Auditor does not simply rank genes. It identifies contradictions across literature, dependency, patient evidence, survival, gene role, pathway, and therapeutic relevance layers.

## Limitations to mention

- Research and education only.
- Public-data only.
- Not clinically validated.
- Survival is currently descriptive.
- Curated annotation tables are transparent but incomplete.
